# Codigo fuente sugerido

Archivos utiles cuando pasen del notebook al proyecto:

- `data_utils.py`: carga de captions y splits.
- `preprocess.py`: limpieza, tokenizacion y padding.
- `features.py`: extraccion de features de la CNN.
- `model.py`: encoder, decoder y compilacion.
- `train.py`: entrenamiento y guardado.
- `inference.py`: generacion de captions sobre imagenes nuevas.
